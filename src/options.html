OpenAI

Get a key at platform.openai.com/api-keys. Suggested: gpt-5.4-nano (cheap), gpt-5-mini (better OCR).

Anthropic

Get a key at console.anthropic.com. Suggested: claude-haiku-4-5 (fast, excellent OCR).

Google (Gemini)

Get a key at aistudio.google.com. Suggested: gemini-3.1-flash-lite-preview (cheapest, fast).

Ollama (local)

Run a vision model first: ollama pull gemma4:e2b (or qwen2-vl, llava, llama3.2-vision). Start Ollama with extension origin allowed:
OLLAMA_ORIGINS=chrome-extension://* ollama serve

Prompt

The instruction sent to the model alongside the image. Tweak this if you want different style — e.g. shorter output, different language, transcription only.
